# Exa MCP 연동

## 작성시각
2025년 12월 16일 화요일

## 해결하고자 한 문제
- Exa MCP 서버 연동
- API 키 영구 설정 적용
- Claude Code에서 Exa 검색 기능 사용 가능하도록 설정

## 해결된 것

### 1. Exa MCP 서버 설정 확인
- 기존에 이미 Exa MCP 서버가 `.claude.json` 파일에 등록되어 있었음
- 서버 타입: stdio
- 실행 명령어: `npx -y @exa-labs/exa-mcp-server`

### 2. API 키 영구 설정
- **설정 파일**: `C:\Users\tlduf\.claude.json`
- **변경 전**: `"EXA_API_KEY": "${EXA_API_KEY}"` (환경 변수 참조)
- **변경 후**: `"EXA_API_KEY": "779011b0-4629-4c8e-a5cb-5436f760323f"` (직접 값 설정)
- 이제 API 키가 영구적으로 설정되어 있으며, 별도의 환경 변수 설정 없이 사용 가능

### 3. MCP 서버 구성
현재 `.claude.json`에 등록된 MCP 서버들:
- **sequential-thinking**: 순차적 사고 지원 서버
- **exa**: 웹 검색 및 컨텐츠 검색 서버 (신규 API 키 설정 완료)

## 해결되지 않은 것
- 특별히 해결되지 않은 문제 없음
- Exa MCP 서버가 정상적으로 작동하는지 테스트 필요

## 향후 개발을 위한 컨텍스트

### Exa MCP 기능
Exa MCP 서버는 다음과 같은 기능을 제공합니다:
- 웹 검색 기능
- 컨텐츠 검색 및 추출
- AI 어시스턴트와 검색 엔진 통합

### 사용 방법
1. **Claude Code 재시작**: 변경 사항 적용을 위해 Claude Code를 재시작
2. **MCP 서버 확인**: `c mcp list` 명령어로 Exa 서버가 정상 연결되었는지 확인
3. **Exa 사용**: Claude Code에서 웹 검색이나 정보 조회 시 자동으로 Exa MCP 서버 활용

### 설정 파일 구조
```json
{
  "mcpServers": {
    "exa": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@exa-labs/exa-mcp-server"],
      "env": {
        "EXA_API_KEY": "779011b0-4629-4c8e-a5cb-5436f760323f"
      }
    }
  }
}
```

### API 키 보안
- API 키가 설정 파일에 평문으로 저장되어 있음
- `.claude.json` 파일은 사용자 홈 디렉토리에 저장되므로 적절한 파일 권한 관리 필요
- Git에 커밋하지 않도록 주의 (사용자 설정 파일이므로 기본적으로 제외됨)
