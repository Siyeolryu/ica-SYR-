# Claude Code 필수 명령어 연습 가이드

## 📋 연습 목적
이 문서는 Claude Code(Cursor)의 필수 명령어를 실습하고 스크린샷을 촬영하기 위한 가이드입니다.

---

## 🎯 연습 방법

각 명령어를 **채팅창에 직접 입력**하고, 실행 결과 화면을 스크린샷으로 저장하세요.

---

## ✅ 필수 명령어 체크리스트

### 1. `/init` - 프로젝트 초기화
**기능**: 프로젝트 초기화, CLAUDE.md 생성

**실습 방법**:
1. 채팅창에 `/init` 입력 후 Enter
2. CLAUDE.md 파일이 생성되는지 확인
3. 스크린샷 저장: `01_init_명령어.png`

**예상 결과**: 프로젝트 정보를 담은 CLAUDE.md 파일 생성

---

### 2. `/cost` - 비용 확인
**기능**: 현재 세션 비용 확인

**실습 방법**:
1. 채팅창에 `/cost` 입력 후 Enter
2. 표시되는 비용 정보 확인
3. 스크린샷 저장: `02_cost_명령어.png`

**예상 결과**: 토큰 사용량 및 비용 표시

---

### 3. `/doctor` - 시스템 진단
**기능**: 설정 및 연결 상태 진단

**실습 방법**:
1. 채팅창에 `/doctor` 입력 후 Enter
2. 시스템 상태 정보 확인
3. 스크린샷 저장: `03_doctor_명령어.png`

**예상 결과**: 환경 설정 및 연결 상태 정보 표시

---

### 4. `/compact` - 컨텍스트 압축
**기능**: 컨텍스트 압축 (토큰 절약)

**실습 방법**:
1. 채팅창에 `/compact` 입력 후 Enter
2. 압축 진행 상황 확인
3. 스크린샷 저장: `04_compact_명령어.png`

**예상 결과**: 대화 기록 압축, 토큰 사용량 감소

---

### 5. `/clean` - 대화 기록 초기화
**기능**: 대화 기록 초기화

**실습 방법**:
1. 채팅창에 `/clean` 입력 후 Enter
2. 확인 메시지 확인
3. 스크린샷 저장: `05_clean_명령어.png`

**⚠️ 주의**: 이 명령어는 **가장 마지막에** 실습하세요! (대화 기록이 삭제됩니다)

---

### 6. `/help` - 도움말
**기능**: 도움말 확인

**실습 방법**:
1. 채팅창에 `/help` 입력 후 Enter
2. 표시되는 명령어 목록 확인
3. 스크린샷 저장: `06_help_명령어.png`

**예상 결과**: 사용 가능한 모든 명령어 목록 표시

---

### 7. `/config` - 설정 확인/변경
**기능**: 설정 확인/변경

**실습 방법**:
1. 채팅창에 `/config` 입력 후 Enter
2. 현재 설정 정보 확인
3. 스크린샷 저장: `07_config_명령어.png`

**예상 결과**: 현재 Claude Code 설정 정보 표시

---

### 8. `/terminal-setup` - 터미널 설정
**기능**: 터미널에서 shift+Enter로 다음줄 입력 가능하게 함

**실습 방법**:
1. 채팅창에 `/terminal-setup` 입력 후 Enter
2. 설정 완료 메시지 확인
3. 스크린샷 저장: `08_terminal-setup_명령어.png`

**예상 결과**: 터미널 멀티라인 입력 설정 완료

1. **순서**: `/clean` 명령어는 가장 마지막에 실습하세요
2. **스크린샷**: Windows에서 `Win + Shift + S`로 화면 캡처 가능
3. **확인**: 각 명령어 실행 후 결과를 꼼꼼히 확인하세요
