# 프로젝트 실행 가이드

## 🚀 빠른 시작

### 1단계: Node.js 설치 확인

먼저 Node.js가 설치되어 있는지 확인하세요:

**Windows PowerShell에서 확인:**
```powershell
node --version
npm --version
```

**설치되어 있지 않다면:**
1. [Node.js 공식 웹사이트](https://nodejs.org/)에서 LTS 버전 다운로드
2. 설치 후 터미널 재시작

### 2단계: 의존성 설치

프로젝트 폴더에서 다음 명령어를 실행하세요:

```bash
npm install
```

이 명령어는 `package.json`에 있는 모든 라이브러리를 설치합니다:
- Next.js 14
- React 18
- TailwindCSS
- Recharts (차트 라이브러리)
- TypeScript

### 3단계: 개발 서버 실행

```bash
npm run dev
```

성공하면 다음과 같은 메시지가 표시됩니다:
```
  ▲ Next.js 14.0.4
  - Local:        http://localhost:3000
  - Ready in 2.3s
```

### 4단계: 브라우저에서 확인

브라우저를 열고 다음 주소로 이동하세요:
```
http://localhost:3000
```

## 📱 확인할 수 있는 기능

### 메인 대시보드 (`http://localhost:3000`)
- ✅ 오늘의 화제 종목 카드 (큰 카드)
- ✅ 주가 차트 (최근 5일간 라인 차트 + 거래량 바 차트)
- ✅ 선정 기준 표시 (거래량 상위, 상승률 상위)
- ✅ 최근 브리핑 히스토리
- ✅ 브리핑 생성 버튼

### 브리핑 상세 페이지 (`http://localhost:3000/briefings/[id]`)
- ✅ 브리핑 이미지 미리보기
- ✅ 리포트 텍스트
- ✅ 이메일/슬랙 발송 버튼
- ✅ 포함된 종목 정보

## 🛠️ 문제 해결

### npm이 인식되지 않는 경우

**Windows에서 npm 설치 확인:**
1. Node.js 설치 확인: `node --version`
2. 환경 변수 확인 필요 시 Node.js 재설치

**대안: Yarn 사용**
```bash
# Yarn 설치 (npm이 없는 경우)
npm install -g yarn

# 의존성 설치
yarn install

# 개발 서버 실행
yarn dev
```

### 포트 3000이 이미 사용 중인 경우

다른 포트로 실행:
```bash
npm run dev -- -p 3001
```

### 빌드 오류가 발생하는 경우

캐시 삭제 후 재시도:
```bash
# .next 폴더 삭제
rm -rf .next

# Windows PowerShell
Remove-Item -Recurse -Force .next

# 재설치 및 실행
npm install
npm run dev
```
