# 서비스 기획서: 당신이 잠든 사이 (While You Were Sleeping)

## 1. 서비스 개요

"당신이 잠든 사이"는 미국주식 투자하는 한국인 직장인을 위한 완전 자동화된 데일리 브리핑 서비스입니다. 한국 시간 기준 매일 아침 7시, Yahoo Finance의 거래량과 상승률 데이터를 분석하여 가장 활발하게 움직인 상위 5개 화제 종목을 자동으로 선정합니다. 선정된 종목의 기본 정보와 Exa API를 통해 수집한 관련 뉴스를 Gemini API로 요약하고, 종목이 화제가 된 원인을 분석하여 AI가 생성한 이미지 브리핑으로 제공합니다. 복잡한 분석이나 주관적 판단 없이 객관적 데이터 기반으로 "가장 활발한 종목"만 자동 선정하여 제공하며, 사용자의 입력 없이 완전 자동으로 작동하는 Chatless 경험이 핵심 차별점입니다. 생성된 브리핑은 슬랙으로 자동 발송되며, 로컬에도 저장되어 언제든지 확인할 수 있습니다.

## 2. 핵심 가치 제안 (Value Proposition)

### "복잡한 분석 없이, 가장 활발한 종목만 자동으로"

**핵심 메시지**: 미국주식 투자에 필요한 정보를 한국 시간 기준 매일 아침 7시 자동으로 수집하고 분석하여, 복잡한 분석 없이 핵심만 빠르게 파악할 수 있는 완전 자동화 서비스

**차별점**:
- **복잡한 분석 없이 "가장 활발한 종목" 자동 선정**: Yahoo Finance의 거래량과 상승률 데이터를 기반으로 가장 주목받는 종목을 객관적 지표로 자동 선정하여, 주관적 판단이나 복잡한 분석 과정 없이 핵심 정보만 제공
- **Chatless (입력 없이 자동)**: 사용자가 별도로 검색하거나 질문할 필요 없이, 한국 시간 기준 매일 아침 7시 자동으로 필요한 정보만 선별하여 슬랙으로 전달하는 완전 자동화된 서비스

**주요 가치**:
- **완전 자동화**: 한국 시간 기준 매일 아침 7시 자동 실행되어 화제 종목 선정부터 브리핑 생성 및 슬랙 발송까지 모든 과정이 자동으로 진행되며, 사용자의 개입 없이 필요한 정보만 선별하여 제공
- **AI 기반 분석**: Exa API로 수집한 뉴스와 Gemini API를 활용하여 종목이 화제가 된 원인을 자동 분석하고, 종목 정보와 뉴스를 요약한 이미지 브리핑으로 제공
- **시간 효율성**: 바쁜 직장인들이 한국에서 잠자는 동안 미국 증시에서 일어난 중요한 변화를 출근 전 짧은 시간에 빠르게 파악할 수 있도록 슬랙으로 한눈에 보기 쉬운 이미지 브리핑을 제공

## 3. 주요 기능 목록

1. **자동 화제 종목 선정**
   - Yahoo Finance API를 통한 실시간 거래량 및 상승률 데이터 수집
   - 거래량과 상승률을 종합한 알고리즘 기반 화제 종목 자동 선정
   - 상위 5개 종목 선정 (가장 활발한 종목)

2. **종목 정보 수집 및 정리**
   - 선정된 종목의 기본 정보 수집 (현재가, 전일 대비, 시가총액 등)
   - 회사 기본 정보 요약

3. **뉴스 수집 및 요약**
   - Exa API를 통한 종목 관련 최신 뉴스 자동 수집
   - Gemini API를 활용한 뉴스 핵심 내용 요약 및 분석

4. **AI 이미지 브리핑 생성**
   - Gemini API를 사용한 브리핑 텍스트 생성
   - 종목 정보, 뉴스 요약, 화제 원인 분석을 포함한 시각적 브리핑 이미지 자동 생성
   - 한눈에 파악 가능한 레이아웃으로 구성

5. **자동 실행 및 슬랙 발송**
   - 한국 시간 기준 매일 아침 7시 자동 실행 (APScheduler 사용)
   - 생성된 브리핑 이미지를 슬랙 채널로 자동 발송
   - 브리핑 이미지와 데이터를 로컬에 자동 저장하여 과거 히스토리 조회 가능

## 4. 사용자 시나리오 (User Journey)

### 시나리오: 김대리 (32세, IT 기업 직장인, 미국주식 투자 경력 2년)

**1단계: 서비스 발견 및 가입 (Day 0)**
- 미국주식 커뮤니티에서 "당신이 잠든 사이" 서비스 소개글 발견
- 복잡한 분석 없이 핵심 종목만 자동으로 받을 수 있다는 점에 관심
- 간단한 이메일 주소 입력으로 가입 완료
- 슬랙 워크스페이스 연동 선택 (업무용 슬랙 사용 중)

**2단계: 첫 브리핑 수신 (Day 1)**
- 한국 시간 기준 오전 7시, 슬랙 채널에 첫 브리핑 이미지 수신
- 전날 밤 미국 증시에서 가장 활발했던 5개 종목이 한눈에 정리되어 있음
- 각 종목의 현재가, 전일 대비, 핵심 뉴스 요약 확인
- 출근 준비 중 스마트폰으로 빠르게 훑어봄 (약 2분 소요)

**3단계: 일상적인 사용 (Week 1-4)**
- 매일 아침 슬랙으로 브리핑 수신
- 브리핑에서 관심 있는 종목 발견 시, 상세 분석은 별도로 진행
- 주말에는 이메일로 주간 요약 브리핑 수신
- 특정 종목에 대한 알림 설정 (해당 종목이 화제 종목에 포함될 때)

**4단계: 서비스 활용 심화 (Month 2+)**
- 브리핑을 통해 놓쳤을 종목 동향 파악
- 과거 브리핑 아카이브를 통해 특정 종목의 이전 동향 추적
- 동료 투자자들에게 서비스 추천
- 브리핑 내용을 바탕으로 투자 의사결정에 참고

**5단계: 장기 사용 및 충성도 형성 (Month 3+)**
- 매일 아침 브리핑 확인이 루틴화됨
- 서비스 없이는 미국 증시 동향 파악이 불안함
- 프리미엄 기능(더 많은 종목, 상세 분석 등) 유료 구독 고려

## 5. 성공 지표 (KPI)

1. **DAU/MAU (Daily/Monthly Active Users)**
   - **정의**: 일일 활성 사용자 수 / 월간 활성 사용자 수
   - **측정**: 매일 브리핑을 열어본 사용자 수 (DAU), 한 달 동안 최소 1회 이상 브리핑을 확인한 사용자 수 (MAU)
   - **목표**: DAU/MAU 비율 40% 이상 (높은 일일 참여도)

2. **브리핑 오픈률 (Open Rate)**
   - **정의**: 발송된 브리핑 대비 실제 열람한 비율
   - **측정**: 이메일 오픈률, 슬랙 메시지 클릭률
   - **목표**: 이메일 오픈률 30% 이상, 슬랙 메시지 클릭률 50% 이상

3. **사용자 유지율 (Retention Rate)**
   - **정의**: 가입 후 일정 기간 동안 서비스를 지속 사용하는 비율
   - **측정**: 가입 후 7일, 30일, 90일 유지율
   - **목표**: 7일 유지율 60% 이상, 30일 유지율 40% 이상

4. **서비스 추천 지수 (NPS - Net Promoter Score)**
   - **정의**: 사용자가 서비스를 다른 사람에게 추천할 의향을 측정하는 지표
   - **측정**: 0-10점 척도로 추천 의향 조사, NPS = (추천자 비율 - 비추천자 비율) × 100
   - **목표**: NPS 50 이상

5. **종목 선정 정확도 (Relevance Score)**
   - **정의**: 선정된 종목이 사용자에게 실제로 유용한 정도
   - **측정**: 사용자 피드백, 브리핑 열람 시간, 종목 상세 조회율 등을 종합
   - **목표**: 사용자 관심 종목 매칭률 70% 이상
